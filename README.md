## Graph PreQL とは？

---

プリキュアシリーズのデータ（プリキュア、シリーズ作品、主題歌）を GraphQL で取得できる API です。GraphQL の学習等にお使いください！（画像データは含まれていませんので、プリキュア図鑑等を作りたい方はご自身で画像を用意してください）。
